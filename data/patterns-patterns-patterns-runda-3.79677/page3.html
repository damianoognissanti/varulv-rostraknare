Jag får lite frågor om §9 så jag tror att det vore bra att jag förtydligar mig:

§9: Om du löser ett chiffer som anger ett villkor, träder villkoret i kraft med omedelbar verkan. Du kan välja att avstå från att lösa ett chiffer för att undvika detta. SL meddelar ej publikt att villkoret gäller, men du kan om du vill meddela andra det i speltråden.

Paragrafen innebär att ett villkor som aktiverats ej kan återaktiveras eftersom ett chiffer bara kan lösas "en gång". Är det löst så är det löst och så var det med det.

Det som däremot kan inträffa är att ett nytt villkor ersätter det gamla villkoret. Då gäller det nya villkoret. Inte heller då kan det nya villkoret bytas ut mot ett gammalt.

Hoppas detta gav klarhet!
