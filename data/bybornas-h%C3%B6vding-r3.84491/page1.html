Okej tillbaka till ritbordet innan det blir dags för sömn.

Med åtta spelare kan det inte vara fyra ulvar. Nattmord är officiellt. Det borde ge nåt sånt här:

A: Cohortes Praetoriae. Välj ett alternativ:
( ) 0 extra romare från Legio X Equestris. (26 s)
( ) 1 extra romare från Legio X Equestris. (18 s)
( ) 2 extra romare från Legio X Equestris. (10 s)
( ) 3 extra romare från Legio X Equestris. (4 s)

B: Troubadix sång. Välj ett alternativ:
( ) Troubadix sång får romarnas ansikten att bli röda och gallernas gröna. (+6 s)
( ) Troubadix sång får romarnas ansikten att bli gröna och gallernas röda. (+4 s)
( ) Troubadix sång får allas ansikten att bli gröna. (-2 s)
( ) Troubadix sång får allas ansikten att bli röda. (-2 s)

C: Vae victo, vae vicits! Välj så många du har råd med:
( ) Delenda Carthago: Ni får nattmörda på ojämna (R1, R3, o.s.v.) rundor. (-8 s)
( ) Et tu, Brute: Ni får nattmörda på jämna (R2, R4, o.s.v.) rundor. (-8 s)
( ) Caliguliminix har kommit till byn: En romare, ditt val, får alltid grönt ansikte av Troubadix sång. (-8 s)
( ) Aut Caesar, aut nihil: Julius Caesar har en extra röst en gång (skrivs privat till SL). Kan användas för snajp. (-8 s)
( ) Semper Batallius har skapat en tvedräkt: Romarna har ett fritt nattmord att användas när som helst - det förbrukas efter användning. (-8 s)
( ) Druiden har fått en bautasten i huvudet: trolldrycken han delar ut skyddar inte mot nattmord. (-8 s)

Nuläget.

Gröna: Anth, Sidd, Jag.

Vågar inte lita på någon längre, men litar mer på Basse just nu än tidigare. Så bryter min tradition och kommer inte rösta på honom.

Osäkra men misstänkta:

Sysp: Har inte sett så mycket av honom i slutet av runda 2. Tillsammans med Strahd flyttade de sin röst ganska sent till Vimes. Sysp påpekar dock att ulvarna kommer mörda Brynolf vilket även blev utfallet av rundan. Vet inte det säger något.

Strahd: Flyttar sin röst sent till Vimes. Motivet är att ändra turordningen. Det vill säga att Vimes blir utröstad vid ett läge där röstläget är lika (om jag fattar rätt, kan fattat helt fel). Det gör att Vimes löper större risk att bli utröstad. Nu blir Vimes utröstad ändå då lejonparten av rösterna hamnar på honom.

Helt osäkra:

Basse: Jag har ingen aning, men han är garanterat mindre misstänkt än Strahd och Sysp för mig. Basse försvarade även Vimes in i slutet. Det har verkligen varit en lojalitet utan dess like, vilket för mig gör att jag ser Basse som mer grön än röd.

Big Mamma: Jag vet inte längre, dyker upp ibland. Det enda som egentligen talar för grönt här är påståendet om PM:et. Men är det hållbart att grunda ett grönande endast på det? Påverkar så klart även min trovärdighet som grön, då jag hävdat samma sak som ett tecken på att jag är bybo, även om jag inte sagt det rakt ut.

Fjodor: Har ingen aning om Fjodor och var jag ska lägga honom. Känns som jag sett för lite för att säga något bergsäkert.

Jag litar dock inte på Strahd eller Sysp. Så just nu gör jag så här.
