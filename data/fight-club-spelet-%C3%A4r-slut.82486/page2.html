Anthrox said:
Målet med spelet var att se hur dynamiken förändras ifall ulvarna sitter ihop med byborna och måste spela för att rädda vissa bybor, och således ge tydliga signaler i röstmönstret kring vad som händer. Detta i kombination med att dreven segades ner, som egentligen mest var ett artificiellt hinder (Vimes var ju helt rätt ute med att det inte var ett särskilt stort problem att dreva trots allt), var tänkt skapa ett spel där om rätt personer pressas så blir det tydligt var ulvarna kan finnas. Att analysera röstmönster var min bild av nyckeln till hur byn skulle vinna.

Tyvärr så fastnade byn i tråden att försöka utnyttja regelbrottet med att gröna varandra genom detaljer i PM, och istället för att nysta ut tråden så trasslade ni bara in er i den istället. Både Basse och Andersson åkte ut pga upplevda diskrepanser från PM-gate, som i det övriga spelet inte sa något alls om rödning egentligen. Hade ni spelat Vanligt Ulvspel™ (eller för den delen försökt analysera spelet istället ) och släppt PM-spåret hade Basse och Andersson aldrig hamnat under luppen på samma sätt.

Jag funderade större delen av första rundan vad jag skulle göra med regelbrottet (Vimes #20 och Waterhouse #16), men tänkte först att det skulle räcka att posta bybo-PM och låta det fortgå. Jag själv missade då tyvärr detaljen "Fighter" och såg inte dess relevans, vilket blåste upp senare när jag inte var lika närvarande. Och då hade ju skeppet redan seglat för länge sedan. Pandoras ask var öppnad och gick inte att stänga igen. Det var Sidd som 19.54 observerade att byborna pratade om sig själva som Fighters och först då insåg jag mitt misstag. För att släta över obalansen som uppstått (Både Vimes och WH var oklanderligt gröna vid det laget, tekniskt sett) gav jag ulvarna ett extra nattmord (de hade bara ett från början) för att ha en chans att komma över händelsen med två grönade bybor från ett regelbrott, och sen lät jag det vara. Nästa gång kommer jag dock passa mig för att lita på att det räcker med att säga till att inte diskutera PM, och istället bara posta det PM som skickas ut till bybor.

För övrigt så tycker jag att ni borde lyssna på @Brynolf mer, han var snubblande nära med massor av poster att knäcka spelupplägget, men fick aldrig något gehör för dem.
