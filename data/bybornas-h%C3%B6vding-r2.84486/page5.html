Jag redigerade om damogns fröjdiga meny till något som blev lite lättare att läsa för de som vill bryta i den. Det blev enklare att förstå tycker jag om poängen i de första två obligatoriska kategorierna ändrades så att man köper en sämre siare, inte får extrapoäng för en vanlig eller galen.

Jag går vidare och börjar bryta lite i scenarion nu men postar menyn här separat så den inte blandas ihop med mina vilda teorier sen.

Byn har en siare, en väktare och en hövding.
Hövdingen är känd av alla när Ceasar gör sina val och kan inte röstas ut eller nattmördas R1 eller R2.
Caesar får välja ulvar själv.
Poängen får inte vara negativ när alla val är gjorda (=måste sluta på 0 eller högre).

A: Antal ulvar. Välj ett alternativ:
( ) Ensamulv (+32)
( ) Två ulvar (+24)
( ) Tre ulvar (+16)
( ) Fyra ulvar (+10)

B: Siaren. Välj ett alternativ:
( ) Vanlig siare (0)
( ) Galen siare - ulvar sias grönt, bybor sias rött (-2)
( ) Naiv siare - alla sias grönt (-8)
( ) Paranoid siare - alla sias rött (-8)

C: Nattmord. Välj så många du vill och har råd med.
( ) Nattmord ojämna (R1, R3, o.s.v.) rundor. (-8)
( ) Nattmord jämna (R2, R4, o.s.v.) rundor. (-8)
( ) Extra nattmord valfri runda (-8 s)
( ) Värdelös väktare - Väktning skyddar inte mot nattmord (-8 s)

D: Övrigt. Välj så många du vill och har råd med.
( ) Ulvmiller - Valfri ulv sias alltid grönt (-8 s)
( ) Extraröst i valfri runda (-8 s)

Random observationer när jag börjar analysera menyn ordentligt:

1) Allting köps i jämna 8:or utom galen siare (-2) och den fjärde ulven (-6).

Det är med andra ord bättre att välja naiv siare än galen siare i alla fall utom när man är fyra ulvar annars fryser sex poäng inne. När man är exakt fyra ulvar så är det "gratis" att välja galen siare istället för vanlig. Då har man ändå råd med exakt ett val av nattmord, ulvmiller eller extraröst.

2) Paranoid siare är inte vald eftersom Brynolf siade Vimes grönt, så den kan vi stryka.

3) Extrarösten kostar lika mycket som alla andra val och verkar betydligt svagare, så den är sannolikt inte vald.
