Regler:

§0: Spelet består av två fraktioner: bybor och varulvar.

§0a: Byborna vinner när alla varulvar är döda.

§0b: Varulvarna vinner när de lika många som byborna.

§1: Bybor får inte prata om spelet med varandra utanför den aktuella speltråden.

§1a: Varulvarna får prata med varandra utanför speltråden.

§1b: Det är otillåtet att redigera och radera inlägg.

§1c: Det är otillåtet att posta bilder som föreställer roll-PM, eller som föreställer din inbox på WRNU.

§2: En spelare kommer röstas ut varje runda.

§2a: Du får byta kandidat för röstningen hur många gånger som helst. Det är den senaste rösten som räknas.

§2b: Rösterna måste kunna fångas av Martin3:s rösträkningsverktyg för att räknas. Använd formatering med @-tagg:

Röst: @damogn

§3: Varje runda har en deadline som meddelas vid rundans start.

§3a: När den nåtts får inga nya inlägg skrivas i tråden.

§3b: Den som då fått flest röster på sig försvinner ur spel.

§3ba: Om det är oavgjort mellan två eller flera spelare är äldsta röst utslagsgivande.

§3c: Den utröstades fraktion avslöjas i uppstarten av nästkommande runda.


Julius Caesars Imperium (meny):

Caesar får sätta ihop Praetoriangardet själv.
Statskassan måste ha 0 eller fler sestertier när alla val är gjorda.

A: Cohortes Praetoriae. Välj ett alternativ:
( ) 0 extra romare från Legio X Equestris. (26 s)
( ) 1 extra romare från Legio X Equestris. (18 s)
( ) 2 extra romare från Legio X Equestris. (10 s)
( ) 3 extra romare från Legio X Equestris. (4 s)

B: Troubadix sång. Välj ett alternativ:
( ) Troubadix sång får romarnas ansikten att bli röda och gallernas gröna. (+6 s)
( ) Troubadix sång får romarnas ansikten att bli gröna och gallernas röda. (+4 s)
( ) Troubadix sång får allas ansikten att bli gröna. (-2 s)
( ) Troubadix sång får allas ansikten att bli röda. (-2 s)

C: Vae victo, vae vicits! Välj så många du har råd med:
( ) Delenda Carthago: Ni får nattmörda på ojämna (R1, R3, o.s.v.) rundor. (-8 s)
( ) Et tu, Brute: Ni får nattmörda på jämna (R2, R4, o.s.v.) rundor. (-8 s)
( ) Caliguliminix har kommit till byn: En romare, ditt val, får alltid grönt ansikte av Troubadix sång. (-8 s)
( ) Aut Caesar, aut nihil: Julius Caesar har en extra röst en gång (skrivs privat till SL). Kan användas för snajp. (-8 s)
( ) Semper Batallius har skapat en tvedräkt: Romarna har ett fritt nattmord att användas när som helst - det förbrukas efter användning. (-8 s)
( ) Druiden har fått en bautasten i huvudet: trolldrycken han delar ut skyddar inte mot nattmord. (-8 s)

Kul med varulvsspel igen och lite annorlunda upplägg! En grön Anthrox och säker är ju en rejäl fördel i R1 och R2 som jag tycker vi borde kunna spela hem, det vänder ju på klassisk varulvsspelslogik om den oinformerade majoriteten som förlorar mot den informerade minoriteten.

Detta plus ett MU-upplägg (där inte MU-Ceasars val avslöjas under spel) gör det här ju till en rätt unik omgång så det är väl bara att plocka på sig analyshatten.

Elva spelare gör ju att fyra ulvar måste vara hyfsat attraktivt från ett röstningsperspektiv, men det här känns samtidigt som en omgång där man verkligen måste ha nattmord för att kunna hävda sig som ulv (hyfsat lågt spelarantal och tre powerroller i byn) och det ser spontant svårt ut att klämma in nattmord med så många ulvar (ska räkna lite på scenarion senare i eftermiddag när jag har mer tid). Så ett lägre ulvantal (1-3) och fulla nattmord verkar mer troligt.

Frågan är om vi kan nyttja hövding Anthrox på något sätt likt svällskroppssiaren i Valulv för att koordinera röstning så att det blir svårt att gömma sig. Å andra sidan ledde det ju väldigt dålig signal från röstningen så det kanske är bättre att köra VJVS och verkligen tvinga folk att ta ställning istället för att glatt lägga sig där Anthrox pekar.
