Har skrivit kod som ni kan köra i webbläsaren för att se vem som lågflyger:

Hitta personen du vill kolla, t.ex. https://www.rollspel.nu/members/damogn.8892/
Klicka på Postings
Klicka på "See more" längst ner
URL:en är nu t.ex. https://www.rollspel.nu/search/631376/
Det sista är ID för den du ska kolla. (OBS! Detta ändras över tid, så kolla att du har rätt ID för den du ska kolla).
Högerklicka i webbläsaren och ta Inspektera.
Välj fliken Console.
Fyll i ID inom fnuttarna på variabeln person nedan
Fyll i titlarna på URL-erna du söker efter (för oss är det mu6-r1, osv, vilket man ser i adressfältet). Man måste vara försiktig med kommatecken och fnuttar när man expanderar listan. Du behöver inte ändra detta för rundor 1-3.
Fyll i antal sidor bakåt du vill kolla i Postings (10 bör räcka just nu).
Den räknar nu ut hur många poster personen gjort.
OBS! Ifall Chrome inte låter dig klistra in står det var du måste skriva i konsollen för att den ska tillåta det.

var person="631376";
var titlar = ["mu6-r1","mu6-r2","mu6-r3"];
var listaRundor = new Array(titlar.length).fill(0);
var antalsidor = 10;

var request = [];
for (var sida = 1; sida<antalsidor; sida++){
(function(sida){
request[sida] = new XMLHttpRequest();
request[sida].open("GET", "https://www.rollspel.nu/search/"+person+"/?page="+sida, true);
request[sida].onreadystatechange = function() {
if (request[sida].readyState == 4){
for (var t = 0; t<titlar.length; t++){
var titel = titlar[t];
var regexSearch = new RegExp("<a href=\"/threads/"+titel, "g");
console.log(regexSearch)
console.log(((request[sida].responseText || '').match(regexSearch) || []).length);
listaRundor[t]=listaRundor[t]+((request[sida].responseText || '').match(regexSearch) || []).length;
}
}
}
request[sida].send(null);
})(sida);
}


Svaren är nu sparade i variabeln listaRundor. Du kan skriva ut dem genom att (efter det ovan körts) klistra in:

console.log("Spelare " + person + " har skrivit så här många meddelanden varje runda:")
for (var t = 0; t<titlar.length; t++){
console.log(titlar[t]+": " + listaRundor[t])
}

Om Octagonal är galen siare har vi följande givna val som MU tagit :

(X) Ni får nattmörda på ojämna (R1, R3, o.s.v.) rundor (-3)
(X) Ni får nattmörda på jämna (R2, R4, o.s.v.) rundor (-2)
(X) En bybo är galen fraktionssiare (d.v.s. siar ulvar som gröna och bybor som röda) (+1)
(X) Alla siare får sia sig själva eller döda spelare (+1)

Dvs MU ligger på -3 efter dessa val.

Dessa val har MU inte tagit:
( ) 4 Extra ulvar (-4) - poängen räcker inte till
( ) 5 Extra ulvar (-8) - poängen räcker inte till
( ) En av kategorierna (A-E) behöver aldrig avslöjas (-1) - avslöjat R2
( ) En av kategorierna (A-E) avslöjas direkt i spelstart (+1) - avslöjat R2

Så här ligger MU till efter att ha valt antal ulvar:
( ) Inga extra ulvar - du är ensam i ulvlaget (+8) : 8-3= 5 poäng kvar att handla för
( ) 1 extra ulv (+4) : 4-3 = 1 poäng kvar att handla för
( ) 2 extra ulvar (+1) : 1-3 = -2 poäng som måste hittas på annat håll
( ) 3 extra ulvar (+/- 0) : 0-3 = -3 poäng som måste hittas på annat håll

Sista plusposten som MU kan handla förutom att gå ner i ulvantal är:
( ) En bybo är fraktionssiare (d.v.s. siar ulvar som röda och bybor som gröna) (+5)

Så här många poäng har alltså MU kvar att handla för:
Solo - 1 ulv totalt - 5 poäng utan äkta siare, 10 poäng med äkta siare
Duo - 2 ulvar totalt - 1 poäng utan äkta siare, 6 poäng med äkta siare
Trio - 3 ulvar totalt - Måste ta med äkta siare så 3 poäng
Kvartett - 4 ulvar totalt - Måste ta med äkta siare så 2 poäng

MU kan använda poängen på följande lista och får sluta på plus men inte på minus:
( ) Du får välja vilka dina ulvkompanjoner är (-1)
( ) En bybo är naiv fraktionssiare (d.v.s. siar alla som gröna) (+/- 0)
( ) En bybo är paranoid fraktionssiare (d.v.s. siar alla som röda) (+/- 0)
( ) En ulv, ditt val, sias alltid som grön (-2)
( ) Du har, som Mäster Ulv, en extra röst (-2)
( ) Ni har ett fritt nattmord att användas när som helst - det förbrukas efter användning (-3)

So far bara fakta än som baserar sig på scenariot att Octa och jag är gröna. Är det inte så så blir analysen fel. Ska börja tugga igenom scenariona med 1-4 ulvar nu och ge min egen syn på vad som är sannolikt och inte.
